## 1. เริ่มต้นใช้งาน

### ค้นหา Bot

เปิดแอป Telegram แล้วค้นหา Bot ของเราด้วยชื่อ username ที่ตั้งไว้ หรือคลิกลิงก์ที่ได้รับจากระบบ

### พิมพ์ /start

พิมพ์ `/start` เพื่อเริ่มต้น Bot จะถามให้คุณ **แชร์เบอร์โทรศัพท์** ผ่านปุ่มที่แสดงขึ้นมา

> กดปุ่ม "Share Contact" หรือ "แชร์เบอร์โทร" เพื่อส่งเบอร์โทรศัพท์ให้ Bot

---

## 2. สมัครสมาชิก / เชื่อมต่อบัญชี

หลังจากแชร์เบอร์โทรแล้ว จะมี 2 กรณี:

### กรณี A: มีบัญชีอยู่แล้ว (สมัครผ่านเว็บไซต์มาก่อน)

Bot จะพบบัญชีที่ผูกกับเบอร์โทรนี้ แล้วถามรหัสผ่าน:

```
พบบัญชีที่ลงทะเบียนด้วยเบอร์นี้แล้ว
กรุณาใส่รหัสผ่านเพื่อเชื่อมต่อบัญชี:
```

พิมพ์รหัสผ่านที่ใช้สมัครบนเว็บ → เชื่อมต่อสำเร็จ พร้อมใช้งาน

### กรณี B: ยังไม่มีบัญชี (สมัครใหม่ผ่าน Bot)

Bot จะพาผ่านขั้นตอนสมัครทีละขั้น:

| ขั้นตอน | สิ่งที่ต้องทำ                                | ตัวอย่าง               |
| -------------- | --------------------------------------------------------- | ------------------------------ |
| 1              | ใส่ชื่อ-นามสกุล                             | `สมชาย ใจดี`        |
| 2              | ใส่วันเกิด (DD/MM/YYYY)                         | `15/06/1985`                 |
| 3              | เลือกเพศ (กดปุ่ม)                           | ชาย / หญิง / อื่นๆ |
| 4              | เลือกบทบาท (กดปุ่ม)                       | ผู้ป่วย / แพทย์    |
| 5              | ตั้งรหัสผ่าน (8 ตัวอักษรขึ้นไป) | `MyP@ss1234`                 |

> **สำหรับแพทย์:** ระบบจะตรวจสอบใบอนุญาตกับแพทยสภาอัตโนมัติ

สมัครเสร็จ → บัญชีถูกสร้างและเชื่อมต่อกับ Telegram ทันที พร้อมใช้งาน

### กรณี C: เชื่อมต่อผ่านลิงก์จากเว็บ

หากคุณสมัครผ่านเว็บแล้ว ไปที่หน้า Settings บนเว็บ → กดปุ่ม "เชื่อมต่อ Telegram" → ระบบจะสร้างลิงก์ให้คลิก → เปิดใน Telegram → เชื่อมต่อสำเร็จโดยไม่ต้องใส่รหัสผ่าน

## 3. บันทึกค่าความดันโลหิต

มี 3 วิธี:

### วิธีที่ 1: Mini App (แนะนำ)

พิมพ์ `/bp` → กดปุ่ม **"📝 บันทึกความดัน"** → เปิด Mini App ใน Telegram

Mini App แสดง:
- ฟอร์มบันทึกค่า SYS/DIA/PULSE + ปุ่มถ่ายรูป (OCR)
- สถิติเฉลี่ย + ระดับความดันตามเกณฑ์ AHA
- 5 รายการล่าสุด
- สถิติขั้นสูงสำหรับ Premium (SD, PP, MAP, Trend)

> Mini App จะปรากฏเป็นปุ่มหลัง `/start` และ `/stats` ด้วย

### วิธีที่ 2: ถ่ายรูปเครื่องวัด

**ขั้นตอน:**

1. **ถ่ายรูป** หน้าจอเครื่องวัดความดันโลหิต
2. **ส่งรูป** ให้ Bot (ส่งเป็นรูปภาพ หรือส่งเป็นไฟล์ก็ได้)
3. Bot จะวิเคราะห์รูปด้วย AI แล้วแสดงค่าที่อ่านได้:

```
ฉันอ่านได้:
Sys: 120
Dia: 80
Pulse: 72
วันที่: 2026-03-15
เวลา: 14:30

ถูกต้องไหม?
[✅ ถูกต้อง] [✏️ แก้ไข]
```

4. **ถ้าค่าถูกต้อง** → กดปุ่ม "✅ ถูกต้อง" → บันทึกเรียบร้อย
5. **ถ้าค่าไม่ถูกต้อง** → กดปุ่ม "✏️ แก้ไข" → พิมพ์ค่าที่ถูกต้องเอง (ดูวิธีที่ 2)

> ⏳ **ระบบบันทึกอัตโนมัติ (Auto-Save):** หากคุณส่งรูปแล้วลืมกดปุ่มยืนยัน หรือปิดแชทไป ระบบจะถือว่าค่าถูกต้องและ **บันทึกข้อมูลให้อัตโนมัติภายใน 2 นาที** เพื่อป้องกันข้อมูลสูญหาย

**รูปแบบไฟล์ที่รองรับ:** JPEG, PNG

**เคล็ดลับถ่ายรูปให้ AI อ่านได้แม่นยำ:**

- ถ่ายตรงๆ ไม่เอียง ไม่มีเงาบัง
- ให้เห็นตัวเลขชัดเจนทั้ง 3 ค่า (SYS, DIA, Pulse)
- หน้าจอเครื่องวัดสว่างพอ ไม่มืดเกินไป

### วิธีที่ 3: พิมพ์ค่าเอง

หากกดปุ่ม "✏️ แก้ไข" หลัง AI อ่านค่า ให้พิมพ์ค่าในรูปแบบ:

```
SYS/DIA PULSE
```

**ตัวอย่าง:**

```
120/80 72
```

ความหมาย: ความดันบน 120, ความดันล่าง 80, ชีพจร 72

**รูปแบบอื่นที่ใช้ได้:**

- `120 80 72` (ใช้เว้นวรรค — แนะนำที่สุด ง่ายไม่ต้องสลับแป้น)
- `120/80 72` 
- `120/80/72`
- `120-80-72`

## 4. ดูสถิติและกราฟ

### พิมพ์ /stats

Bot จะแสดง:

**1) สถิติเฉลี่ย + ระดับความดัน**

```
ข้อมูลสุขภาพของคุณ
ค่าเฉลี่ย (คำนวณจาก 10 รายการ):
120/80 mmHg (ชีพจร 72)
🟢 ระดับความดัน: ปกติ
```

ระดับความดันตามเกณฑ์ AHA/ACC 2017:

| ระดับ | SBP | DBP | สัญลักษณ์ |
|-------|-----|-----|-----------|
| ปกติ (Normal) | < 120 | < 80 | 🟢 |
| สูงเล็กน้อย (Elevated) | 120-129 | < 80 | 🟡 |
| ความดันสูงระยะที่ 1 | 130-139 | 80-89 | 🟠 |
| ความดันสูงระยะที่ 2 | >= 140 | >= 90 | 🔴 |
| วิกฤต (Crisis) | > 180 | > 120 | 🚨 |

**2) สถิติขั้นสูง (Premium เท่านั้น)**

```
สถิติขั้นสูง (Premium)
• SD: 120 ± 8.5 / 80 ± 6.2 mmHg
• Pulse Pressure: 40 mmHg
• MAP: 93 mmHg
• แนวโน้ม: 📈 เพิ่มขึ้น (+0.5 mmHg/วัน)
```

**สูตรคำนวณ:**

| ค่า | สูตร | ความหมาย |
|-----|-------|----------|
| **ค่าเฉลี่ย (Mean)** | `sum(values) / n` | ค่ากลางทั้งหมด |
| **SD (Standard Deviation)** | `sqrt(sum((x - mean)^2) / (n-1))` | ความเสถียรของค่าความดัน — SD สูง = ความดันแกว่งมาก |
| **Pulse Pressure (PP)** | `avg_SBP - avg_DBP` | PP > 60 = หลอดเลือดแข็งตัว |
| **MAP (Mean Arterial Pressure)** | `(avg_SBP + 2 × avg_DBP) / 3` | MAP >= 65 = เลือดไหลเวียนเพียงพอ |
| **CV (Coefficient of Variation)** | `(SD / Mean) × 100` (%) | เปรียบเทียบความแปรปรวนระหว่างบุคคล |
| **Median** | ค่ากลางเมื่อเรียงลำดับ | ไม่ไวต่อค่าผิดปกติ (outlier) |
| **Trend (แนวโน้ม)** | Linear Regression slope | slope > 0 = ความดันเพิ่มขึ้นเรื่อยๆ |

**3) รายการบันทึกล่าสุด**

```
รายการล่าสุด: (10/25)
- 15/03/2026 14:30: 120/80 (72)
- 14/03/2026 10:15: 118/78 (70)
...
```

**4) ปุ่มลัด**

ท้าย /stats จะมีปุ่ม:
- **📝 บันทึกความดัน** → เปิด Mini App
- **🌐 Dashboard** → เปิดเว็บ Dashboard

**5) กราฟแนวโน้มความดันโลหิต**

หากมีข้อมูล 2 รายการขึ้นไป Bot จะส่ง **รูปกราฟ** อัตโนมัติ แสดง:

- เส้นสีแดง: ความดันบน (Systolic)
- เส้นสีน้ำเงิน: ความดันล่าง (Diastolic)
- เส้นสีเขียวประ: ชีพจร (Pulse)
- โซนสีแดงอ่อน: ช่วงความดันสูง (> 140)
- โซนสีฟ้าอ่อน: ช่วงความดันล่างสูง (90-140)
- ตัวเลข SYS/DIA กำกับทุกจุด
- ค่า Pulse กำกับใต้จุดเขียว

## 5. ตั้งค่า

### พิมพ์ /settings

Bot จะแสดงการตั้งค่าปัจจุบัน:

```
⚙️ ตั้งค่า

🌐 ภาษา: ไทย
🕐 เขตเวลา: Asia/Bangkok

[🌐 ภาษา] [🕐 เขตเวลา]
```

### เปลี่ยนภาษา

กดปุ่ม "🌐 ภาษา" → เลือก:

- 🇬🇧 English
- 🇹🇭 ไทย

ข้อความทั้งหมดจะเปลี่ยนเป็นภาษาที่เลือกทันที

### เปลี่ยนเขตเวลา

กดปุ่ม "🕐 เขตเวลา" → เลือกจากรายการ เช่น:

- Asia/Bangkok (GMT+7)
- Asia/Tokyo (GMT+9)
- America/New_York (GMT-5)
- Europe/London (GMT+0)

> เขตเวลาส่งผลต่อวัน-เวลาที่บันทึกในแต่ละรายการ

## 6. อัปเกรดเป็น Premium

### ความแตกต่างระหว่าง Free กับ Premium

| ฟีเจอร์                     | Free                             | Premium               |
| ---------------------------------- | -------------------------------- | --------------------- |
| บันทึกค่าความดัน   | ✅                               | ✅                    |
| ดูสถิติ + กราฟ          | ✅ (30 รายการล่าสุด) | ✅ (ไม่จำกัด) |
| สถิติขั้นสูง (SD, PP, MAP, Trend) | ❌                               | ✅                    |
| ส่งออกข้อมูล (CSV/PDF) | ❌                               | ✅                    |
| ประวัติทั้งหมด       | ❌ (30 รายการล่าสุด) | ✅ ไม่จำกัด   |

### วิธีอัปเกรด

1. พิมพ์ `/upgrade`
2. เลือกแพ็กเกจ:
   - รายเดือน: 9 บาท
   - 12 เดือน: 99 บาท
3. Bot จะแสดง **ข้อมูลบัญชีธนาคาร** สำหรับโอนเงิน
4. โอนเงินตามจำนวนที่แสดง
5. **ส่งรูปสลิปการโอน** ให้ Bot
6. ระบบตรวจสอบสลิปอัตโนมัติ
7. อัปเกรดสำเร็จ → ใช้งาน Premium ได้ทันที

### ตรวจสอบสถานะสมาชิก

พิมพ์ `/subscription` เพื่อดูสถานะปัจจุบัน:

```
💎 สมาชิก Premium
หมดอายุ: 15/04/2026
เหลืออีก: 31 วัน
```

## 8. คำถามที่พบบ่อย

### ส่งรูปแล้ว Bot อ่านค่าผิด ทำอย่างไร?

กดปุ่ม "✏️ แก้ไข" แล้วพิมพ์ค่าที่ถูกต้องเอง เช่น `120/80 72`

### ส่งรูปแล้ว Bot อ่านไม่ได้เลย?

- ลองถ่ายรูปใหม่ให้ชัดขึ้น ไม่มีเงาบังตัวเลข
- ถ่ายตรงๆ ไม่เอียงมาก
- ถ้ายังไม่ได้ กดแก้ไขแล้วพิมพ์ค่าเอง

### Bot บอกว่า "Duplicate Record" คืออะไร?

หมายความว่าคุณเคยบันทึกค่าเดียวกัน (วันที่ + เวลา + ค่า SYS/DIA/Pulse) ไปแล้ว ระบบจะไม่บันทึกซ้ำ

### ลืมรหัสผ่าน ทำอย่างไร?

ปัจจุบันยังไม่มีฟีเจอร์รีเซ็ตรหัสผ่านผ่าน Bot กรุณาใช้ฟังก์ชัน "ลืมรหัสผ่าน" บนเว็บไซต์

### ใช้ Bot ได้กี่คน?

1 บัญชี Telegram เชื่อมต่อได้ 1 บัญชีผู้ใช้ ไม่สามารถเชื่อมหลายบัญชีพร้อมกันได้

### ข้อมูลความดันปลอดภัยไหม?

ข้อมูลส่วนตัว (ชื่อ, เบอร์โทร, เลขบัตรประชาชน) ถูกเข้ารหัส (Fernet Encryption) ก่อนจัดเก็บในฐานข้อมูล แม้แต่ผู้ดูแลระบบก็ไม่สามารถอ่านข้อมูลเหล่านี้ได้โดยไม่มีคีย์

### กราฟแสดงภาษาไทยได้ไหม?

ได้ ถ้าคุณตั้งค่าภาษาเป็น "ไทย" กราฟจะแสดงหัวข้อและ legend เป็นภาษาไทยอัตโนมัติ

### ต้องการยกเลิก Premium ทำอย่างไร?

Premium จะหมดอายุอัตโนมัติตามระยะเวลาที่ซื้อ ไม่มีการต่ออายุอัตโนมัติ เมื่อหมดอายุจะกลับเป็น Free โดยอัตโนมัติ (ข้อมูลไม่สูญหาย แค่ดูได้ 30 รายการล่าสุด)
